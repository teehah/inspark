# InSpark

リアルな花火をWebGLで再現する3Dシミュレータ。ドローン視点で花火の中を飛び回れます。

## 特徴

- **13種類の花火** — 牡丹・菊・柳・冠菊・錦冠菊・銀冠菊・芯入・八重芯菊・千輪・十字・椰子・ダリア・段咲き
- **GPU物理演算** — 重力・空気抵抗・トレイルをシェーダーで解析的に計算
- **ドローンカメラ** — 花火の中を飛び回り、最も見応えのある花火を自動追尾 (MVFスコアリング)
- **ジョグダイヤル** — 任意の種類の花火を選んで打ち上げ
- **月・地面・霧** — 空間認識のための環境要素

## 操作

| 操作 | 説明 |
|------|------|
| ジョグダイヤル (右) | スクロール/タップで花火を選んで打ち上げ |
| 飛ぶボタン (下) | ドローンカメラの離陸/着陸 |
| キャンバスクリック | ランダムな花火を打ち上げ |
| マウスドラッグ | カメラ回転 (地上時) |

## 開発

```bash
npm install
npm run dev
```
